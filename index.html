About

This page holds the results produced with the first iteration of a simple ray tracer developed for a graduate course on Rendering at Drexel University.

Assignment #3 - Acceleration Structure

The acceleration structure implemented was Bounding Volume Hierarchy, using a binary tree with Axis Aligned Bounding Boxes.

The scene description file can now have an accelerator node under the scene node to specify the use of the BVH structure. Otherwise, no acceleration is used. The node can also specify how many objects should be on the leaf, like so:
<accelerator type="bvh" leafSize="1"/>
Type is ignored, since only BVH is implemented.

Assignment #1

Coordinate system is left-handed.

Primitives supported are spheres, partial spheres and triangle meshes.

Bounding boxes are partially implemented but are not used.

Scene is described in an XML file format that is a much simpler subset of the Mitsuba file format.
