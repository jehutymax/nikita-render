Nikita Render

Renderer developed for the Advanced Rendering Techniques course at Drexel.

View project on GitHub

About

This page holds the results produced with the first iteration of a simple ray tracer developed for a graduate course on Rendering at Drexel University.

Assignment #1

Coordinate system is left-handed.

Primitives supported are spheres, partial spheres and triangle meshes.

Bounding boxes are partially implemented but are not used.

Scene is described in an XML file format that is a much simpler subset of the Mitsuba file format.
